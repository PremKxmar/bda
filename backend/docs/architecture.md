## Component Details

### 1. Data Layer

#### Raw Data
- **Source**: NYC TLC Yellow Taxi Trip Records
- **Format**: CSV files
- **Size**: ~7 GB (4 months of data)
- **Records**: ~46 million trips

#### Processed Data
- **Format**: Parquet (columnar, compressed)
- **Contents**:
  - Cleaned trip records
  - Grid cell assignments
  - Feature vectors for ML

#### ML Models
- **Algorithm**: Random Forest Classifier
- **Target**: Congestion level (low/medium/high)
- **Features**: hour, day_of_week, cell_id, avg_speed, vehicle_count

---

### 2. Streaming Layer

#### Kafka Producer
```python
# Replays historical trips as real-time GPS events
{
    "vehicle_id": "taxi_12345",
    "timestamp": "2025-12-06T10:46:30Z",
    "latitude": 40.7128,
    "longitude": -74.0060,
    "speed": 15.5,
    "trip_id": "trip_999"
}
```

#### Kafka Topics
| Topic | Purpose |
|-------|---------|
| `vehicle_positions` | Raw GPS events from producer |
| `dashboard_events` | Processed events for frontend |

#### Spark Structured Streaming
- **Window**: 2-minute tumbling windows
- **Aggregations**: count, avg_speed, std_speed per cell
- **ML Scoring**: Real-time congestion prediction

---

### 3. API Layer

#### REST Endpoints

| Endpoint | Method | Description |
|----------|--------|-------------|
| `/api/health` | GET | Service health check |
| `/api/current-traffic` | GET | Current congestion state |
| `/api/predictions` | GET | ML predictions (15 min) |
| `/api/historical` | GET | Time-series data |
| `/api/hotspots` | GET | Top congested zones |
| `/api/cell/<id>` | GET | Cell-specific details |
| `/api/stats` | GET | Overall statistics |

#### WebSocket
- **Endpoint**: `/ws/stream`
- **Purpose**: Push real-time updates to frontend
- **Rate**: Every 2 seconds

---

### 4. Visualization Layer

#### Landing Page
- Hero section with project title
- Key statistics (data size, trips, accuracy)
- Technology stack showcase
- "Launch Dashboard" CTA

#### Dashboard
- Real-time KPI cards
- Time-series congestion chart
- Top hotspots leaderboard
- Playback controls (play/pause/speed)

#### Kepler.gl Map
- 3D hexagon heatmap
- Moving vehicle points
- Prediction overlay zones
- Interactive tooltips
